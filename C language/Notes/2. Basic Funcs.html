2. 基础语句

2.1 if语句：

// 1 situation
if(表达式1) {}
// 2 situations
if(表达式1) {}
else {}
// more situations
if(表达式1)
else if (表达式2)
else if (表示式3)
....

2.2 switch语句：

switch(表达式)
{	case(常量表达式1)：{} break;
    case(常量表达式2)：{} break;
    ...
    default:{}
}

2.3 while语句：

while(表达式){} //表达式返回0退出循环

2.4 do while语句：

do
{ 
} while(表达式); //先做一次再判断，0退出

2.5 for语句：

for(初始化表达式；退出条件表达式；循环调整表达式)
{}

2.6 break语句：

在循环中跳出一层循环，注意多层循环比较麻烦

2.7 continue语句：

屏蔽后面的操作，跳到下一次循环中

2.8 goto语句：

跳到对应标识符的位置，不建议使用
