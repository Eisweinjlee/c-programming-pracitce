2. 基础语句

if语句：

// 1 situation
if(表达式1) {}
// 2 situations
if(表达式1) {}
else {}
// more situations
if(表达式1)
else if (表达式2)
else if (表示式3)
....

switch语句：

switch(表达式)
{	case(常量表达式1)：{} break;
    case(常量表达式2)：{} break;
    ...
    default:{}
}

while语句：

while(表达式){} //表达式返回0退出循环

do while语句：

do
{ 
} while(表达式); //先做一次再判断，0退出

for语句：

for(初始化表达式；退出条件表达式；循环调整表达式)
{}

break语句：跳出一层循环
continue语句：屏蔽后面的操作，跳到下一次循环中
